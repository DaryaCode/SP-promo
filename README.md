# SubPar Promo Site

Static promo landing page for SubPar.

## Local preview

Open `index.html` directly in a browser.

## Shareable preview

This repo includes a GitHub Pages workflow. After GitHub Pages is enabled for the repo, pushes to `main` publish the site here:

https://daryacode.github.io/SP-promo/

## Notes

- The page is a single static HTML file.
- App screenshots and logo assets live in `assets/`.
- Some golf background images are loaded from Unsplash URLs in `index.html`.
